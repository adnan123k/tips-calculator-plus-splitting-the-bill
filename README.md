# tips calculator plus splitting the bill app
an ios app to easily calculate your total bill plus tips for each one.

## supported tips:
* 0%
* 10%
* 20%

## how to run it:
* install xcode
* open Tipsy.xcodeproj in xcode
* run it on your physical device or on your emulator
